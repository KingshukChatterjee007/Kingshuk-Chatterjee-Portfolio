Kingshuk Chatterjee | Personal Portfolio

A modern, interactive personal portfolio website showcasing my journey as a software developer, my technical skills, and featured projects. Built with a focus on smooth animations and a responsive user interface.

🚀 Tech Stack

* **Frontend Framework:** React 18
* **Build Tool:** Vite
* **Animations:** Framer Motion
* **Icons:** Lucide React
* **UI Components:** TagCloud (Interactive skill cloud)
* **Contact Form:** EmailJS

✨ Features

* **Horizontal Scroll Project Vault:** A curated slider showcasing public and private repositories (e.g., KrownFrame, Sys_Logger, FinReport, Krishi Sahyogi) using Framer Motion's scroll and transform hooks.
* **Interactive 3D Skill Cloud:** Visual representation of technical proficiencies.
* **Contact Integration:** Direct messaging capabilities via EmailJS.
* **Responsive Design:** Optimized for both desktop and mobile viewing.
